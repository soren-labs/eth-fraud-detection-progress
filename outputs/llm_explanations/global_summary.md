# LLM Global Summary

## Research Progress Summary

本研究聚焦于以太坊欺诈检测的可解释性分析，主要任务包括二元欺诈风险检测与欺诈类型倾向分析。通过模型优化与对比实验，成功构建了高性能的欺诈检测系统，并采用SHAP方法对模型决策过程进行解释，实现了检测性能与可解释性的平衡。

## Model Result Summary

优化后的XGBoost模型在欺诈检测任务中表现优异，准确率达96.95%，F1分数为0.9306，ROC-AUC高达0.9923。与随机森林（F1:0.9046）和基准XGBoost（F1:0.9089）相比，优化模型在各项指标上均有显著提升，特别是召回率达到92.20%，表明模型对欺诈交易的识别能力较强。

## Interpretability Summary

SHAP特征重要性分析显示，'唯一收款地址数量'和'首末交易时间差'是影响欺诈判断的最重要特征，SHAP值均超过1.0。交易金额相关特征（平均收款值、总收款以太币）以及交易频率特征（平均交易间隔、发送交易次数）也具有较高重要性，这些发现为理解以太坊欺诈行为模式提供了实证依据。
