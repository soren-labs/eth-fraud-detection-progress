# Ethereum Fraud Detection Thesis Progress

This repository contains the progress materials for an undergraduate thesis on interpretable Ethereum fraud detection.

## Included Content

- Research and progress documents in Markdown
- Data cleaning scripts and reports
- Baseline and optimized model training scripts
- Evaluation results, charts, and comparison reports
- SHAP global and case-level explanation results

## Excluded Content

- Raw thesis Word documents
- Raw source dataset
- Large model binary files
- External third-party projects and local IDE files

## Main Directories

- `docs/`: research notes, chapter planning, and progress documents
- `scripts/`: data cleaning, training, comparison, and SHAP scripts
- `processed/`: cleaned-data metadata and cleaning reports
- `outputs/`: model evaluation reports, charts, and SHAP explanation outputs
